# {{ lockbox-name }} pricing

The cost of {{ lockbox-name }} usage is based on:

* Number of secret versions.
* Number of completed [get](api-ref/Payload/get) operations on a secret.


{% note warning %}

The prices are in effect as of March 10, 2024.
